# RishavendraOS Project Worlds Specification v1.0

## Purpose

Project Worlds are not sections.

Project Worlds are not pages.

Project Worlds are immersive environments representing a specific domain of intelligence.

When a visitor enters a Project World, they should feel as though they are entering the environment where the project exists.

The goal is not to read about the project.

The goal is to experience the project.
